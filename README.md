# AndroidUtilProject

Utility for android app development, e.g. Kotlin extensions, tools etc.

## Library structure

extensions/

* ActivityExt
* BundleExt
* ContextExt
* FileExt
* FragmentExt
* IntentExt
* MapExt
* RxJava3Ext
* UriExt
* ViewExt
* ViewModelExt

utils/

* GoogleMapUtils
* NotificationUtils
* PermissionUtils
